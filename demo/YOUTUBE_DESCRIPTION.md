# YouTube Description (paste-ready)

**Axiom Studio — Physics You Can Actually Touch** 🪐

Most students memorize physics. Almost none get to *experiment* with it. Axiom Studio turns any browser into a real physics lab — no account, no setup, completely free.

Every experiment follows three steps:
🔮 **Predict** — commit to an answer before anything runs
🧪 **Experiment** — adjust the parameters and run a real simulation
💡 **Understand** — meet Axion, an AI tutor that reads your specific numbers and explains exactly why what happened, happened

Under the hood it runs genuine physics engines with RK4 integration and true orbital mechanics — not fake animations.

**Four interactive modules:**
• Projectile Motion (kinematics)
• Pendulum (harmonic motion)
• Collisions (conservation laws)
• Orbital Mechanics in full 3D (gravitation)

🚀 Try it live: https://axiom-studio20.vercel.app
